# 거북목 예방을 위한 데스크 환경 구축 가이드

장시간 개발 환경에서 발생하기 쉬운 거북목 증후군을 예방하기 위해 모니터와 의자 등 데스크 셋업을 최적화하는 기술적 지침을 제공합니다.

## 개요

개발자에게 데스크 환경은 단순한 가구가 아닌 장기적인 생산성을 결정짓는 핵심 인프라입니다. 잘못된 모니터 높이와 의자 설정은 경추 전만(거북목)을 유발하며, 이는 단순 통증을 넘어 집중력 저하와 근골격계 질환으로 이어집니다. 본 문서에서는 올바른 인체공학적 데스크 셋업 방법을 단계별로 설명합니다.

## 1. 모니터 최적화: 시선 처리의 핵심

거북목은 시선이 아래로 향할 때 머리 무게를 지탱하기 위해 목 근육이 과하게 긴장하면서 발생합니다.

### 모니터 높이 및 거리
- **높이**: 모니터 상단 1/3 지점이 눈높이와 일직선이 되도록 조절합니다. 이렇게 하면 화면 중앙을 볼 때 시선이 자연스럽게 약 15~20도 아래를 향하게 되어 눈의 피로도 줄일 수 있습니다.
- **거리**: 눈과 화면 사이의 거리는 팔을 쭉 뻗었을 때 손끝이 화면에 닿을 정도(약 50~70cm)가 적당합니다.
- **각도**: 화면 하단이 사용자 쪽으로 약간 기울어지도록 틸트(Tilt) 기능을 활용하면 가독성이 높아집니다.

### 듀얼 모니터 배치
- **메인 모니터**: 정중앙에 배치하여 고개가 돌아가지 않도록 합니다.
- **보조 모니터**: 사용 빈도가 높다면 메인과 나란히 배치하고, 가끔 참조하는 용도라면 세로(Pivot) 모드로 전환하여 목의 좌우 움직임을 최소화합니다.

## 2. 의자 설정: 90-90-90 법칙

올바른 자세는 하체의 안정성에서 시작됩니다. 아래의 3가지 각도를 기억하세요.

1. **팔꿈치**: 키보드 입력 시 팔꿈치 각도가 **90도** 이상을 유지하며 팔걸이에 편하게 얹어져야 합니다.
2. **골반**: 엉덩이를 의자 깊숙이 밀어 넣고 등받이에 기대었을 때 골반 각도가 **90도**가 되어야 합니다.
3. **무릎**: 발바닥 전체가 지면에 닿은 상태에서 무릎 뒷부분과 의자 끝단 사이에 약간의 공간이 있고 각도가 **90도**여야 합니다.

> 💡 **Tip**: 발이 바닥에 닿지 않는다면 반드시 발 받침대를 사용하여 하중을 분산시키세요.

## 3. 실용적 실천 예제: 휴식 알림 스크립트

완벽한 환경을 구축했더라도 장시간 고정된 자세는 해롭습니다. 50분 코딩 후 10분 휴식을 강제하는 간단한 Python 스크립트를 활용해 보세요.

```python
import time
import ctypes

def break_reminder(minutes):
    while True:
        time.sleep(minutes * 60)
        # Windows 시스템 메시지 박스 알림
        ctypes.windll.user32.MessageBoxW(0, "허리를 펴고 스트레칭을 하세요!", "휴식 시간", 1)

if __name__ == "__main__":
    # 50분마다 알림 실행
    break_reminder(50)
```

## 4. 추가 체크리스트

- [ ] 모니터 암(Monitor Arm)을 사용하여 데스크 공간 확보 및 세밀한 높이 조절
- [ ] 어깨가 위로 솟지 않도록 책상 높이 조절 (일반적으로 70~75cm)
- [ ] 키보드와 마우스는 몸에 최대한 가깝게 배치하여 어깨 벌어짐 방지

## 결론

거북목 예방은 장비의 교체보다 '나에게 맞는 위치'를 찾는 것에서 시작됩니다. 오늘 안내한 가이드에 따라 모니터 암과 의자의 틸트 기능을 다시 설정해 보십시오. 건강한 신체는 유지보수가 불가능한 가장 중요한 하드웨어임을 잊지 마시기 바랍니다.
